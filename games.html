Projects

The Name of this Game is Classified

What is it?

A transmedia game that uses a notebook and the HTC-Vive for a puzzle game. In this game, one player is in the headset and the other has the notebook. The two must then work together to solve puzzles in order for the player in VR to 'escape' from the aliens that have captured them.

My Part

For this project, I designed and modeled the objects in the room as well as did write ups for the project as well as helped deisgn algorithims for the puzzles.

Infinity Lab

What is it?

Infinity Lab is a puzzle game designed for virtual reality. In the game, there are 3 randomized puzzles, designed to test logic, patience, and puzzle solving skills. The game is designed to be used by the HTC Vive.

My Part

For this project, I was in charge of the 3D modeling, texturing, and overall aesthetics, including texturing, lighting, and particle effects.

Emerging Media Lab 3D Models

What is it?

This was a project for an Motion Graphics and Animation class. The project was to chose a room that we spent time in and model it. All of the models were made in Autodesk Maya and rendered through Arnold.
